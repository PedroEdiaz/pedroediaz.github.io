Compilar Archivos .tex

LaTeX es un programa de Hypertexto, esto significa que lo usamos para dar formato a nuestros textos, de la misma forma que HTML le da formato a las páginas Web. La ventaja que tiene aprender usar este programa es que fue pensado desde un inició para escribir documentos matemáticos, así mismo todo el diseño puede ser modificado.

Compilado

Los documentos de LaTeX son de formato .tex. Es necesario instalar el compilador de Latex, en mi caso yo tengo instalado texlive que viene con varios paquetes esenciales. Ahora, para trabajar escribes en un archivo .tex el código y lo compilas en la terminal.



pdflatex file.tex

Esto creará un archivo llamado file.pdf donde se verá plasmado todos los comandos que pusimos, si no estamos acostumbrados a trabajar en la terminal podemos instalar una IDE de LaTeX como Texmaker que trabaja casi igual a una IDE de programación, aprietas un boton y se crea el pdf, también puedes usar una IDE en linea como Overleaf si no estás interesado en descargar programas.

Primer PDF

Ahora que ya sabemos como pasar de tex a pdf ya podemos empezar a escribir código y compilarlo para ver si funciona, Entonces, creamos nuestro documento, escribimos algo ' ...' Compilamos y vemos que nos salé un error, esto es por que debes poner en LaTeX explícitamente donde empieza tu documento y donde termina para eso ponemos este código.


\begin{document}
...
\end{document}

vemos que nos sale una advertencia pero ya tenemos un PDF con texto escrito en el, esta advertencia nos salé por que LaTeX no sabe que tipo de documento quieres, para eso agregamos la siguiente linea de código


\documentclass{article}

Con esta linea le decimos a LaTeX que queremos escribir un articulo, formato que es más que necesario para escribir apuntes y tareas. además podemos darle opciones, entre llaves, al documento, por ejemplo poner la letra a 12 puntos y que el documento sea de 2 columnas.


\documentclass[12pt,twocolumn]{article}

Código extra

Ya con esto tenemos todo para empezar a escribir lo que va adentro de nuestro documento, pero yo agregó dos lineas de código más.


\usepackage[spanish]{babel}
\usepackage[utf8]{inputenc}

Estas lineas de código le dicen a LaTeX que a parte de todo el código interno que tiene le agregue el código de babel con la opción spanish para que el documento esté en español, y traduzca algunos títulos que pone LaTeX automáticamente y de inptenc. la opción de utf8 para que codifique el documento en este estándar.

Resumen

Ya con esto tenemos nuestra primer plantilla con las lineas de código necesarias para compilar nuestros textos. Notamos que los comandos para

LaTeX comienzan con una '\\' y en los puntos suspensivos el texto que queremos que aparezca en nuestro pdf.


\documentclass[12pt]{article}
\usepackage[spanish]{babel}
\usepackage[utf8]{inputenc}
\begin{document}
...
\end{document}

Además podemos compilar nuestro archivo foo.tex, donde foo es el nombre que tiene nuestro archivo, con la terminal en la carpeta del archivo, y ejecutando el siguiente comando podemos compilar nuesto archivo LaTeX en un pdf.



pdflatex foo.tex
