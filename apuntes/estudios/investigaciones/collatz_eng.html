Definitions

We say that the Sucesor of a Natural number \(x\), diferent from \(0\), \(S(x)\), is defined by the Collatz's function. \(\)S(x)=\begin{cases} 3x+1\quad \textrm{if}\quad [x]_2=1\\ x/2\qquad \textrm{if}\quad [x]_2=0\\ \end{cases}\(\) This definition also proof that \(S(x)\) is unique for all \(x\)

We say that a number \(x\) goes to \(x'\) (\(x\mapsto x'\)) if \(x=x'\) or \(S(x)\) goes to \(x'\) \[(x\mapsto x'):=\quad (x=x')\quad\textrm{ or }\quad (S(x)\mapsto x')\]

We say that a number \(n\) is betwen \(x\) and \(x'\) if \(x\mapsto n\) and \(n \mapsto x'\). We also say than \(n\) is properly between \(x\) and \(x'\) If \(n\) is between \(x\) and \(x'\) just as \(n\) is not equal to \(x\) or \(x'\)

If \(x\neq x'\) and \(S(x)\neq x'\) then it always exist \(n\) such that \(n\) is properly between \(x\) and \(x'\)

Proof. If \(x\mapsto x'\) and \(x\neq x'\) then \(S(x)\mapsto x'\) since \(x\mapsto S(x)\) we could choose \(n=S(x)\) hence \(x\neq S(x)\) and by out hipótesis \(S(x)\neq x'\). Q.E.D

If \(x\) goes to \(x'\) and if a predicate, \(P(n)\), is true for \(P(x)\) and \(P(n)\) implies \(P(S(n))\) then \(P(n)\) is true for all \(n\) between \(x\) and \(x'\).

Proof. Since \(x\mapsto x'\) then \(x=x'\) or \(S(x) \mapsto x'\), for the first case if \(P(x)\) is true then our hipotesis is true since \(x\) is between it's self. for the second case we asume that \(P(x)\) is true and since \(x\neq x'\) then \(S(x)\mapsto x'\) since \(S(x)\) is between \(x\) and \(x'\) and \(P(x)\) implies \(P(S(x))\) we can conclude that inducion is a valid tool. Q.E.D

About c

If \(a\geq 1\) then we can describe \(c\), of \(x'\), as above. Were \(B\) is the set of values that \(b\) gets when \(a\) increse. \[c=\sum_{n=0}^{a-1}3^n2^{b_n}\qquad b_{n+1}>b_n \quad \forall b_i\in B\]

Proof. We start by induction under \(a\), If \(a=0\) then \(c=0\), which cannot be describe as a sum of powers. Now if \(a\neq 0\) it exists at least one odd number, \(x\), between \(x\) and \(x'\). since \(n\) is odd we can check that \(S(n)\) show us the behaivour of \(a+1\), now for \(b_i\in B_{x\mapsto n}\) Q.E.D

\(\)\begin{aligned} c_{n+1} &=3(c)+2^b\\ &=3\left(\sum_{i=0}^{a-1} 3^i 2^{b_i} \right) + 2^b\\ &=\sum_{i=0}^{a-1} (3^{i+1}2^{b_i}) + 2^b\\ \end{aligned}\(\) Now we must make a set \(B_{x\mapsto n+1}\), thats

define by \(b_0=b\) and \(b_{i+1}=b'_i\) for all \(b'\in B_{x\mapsto n}\). Now because \(b\) always increase then we can argue that \(b_{i+1}>b_i\) for all \(b\in B\), and conclude the proof. \(\)\begin{aligned} c_{n+1}&=\sum_{i=1}^{a} (3^i 2^{b_{i-1}})+ 3^0 2^{b_0}\\ &=\sum_{i=0}^{a} 3^i 2^{b_i} \end{aligned}\(\) Since \(b_i\) does not reapeat it self and they are well ordered then we can encrypted on a númber that we are gonna call the determinant.

Determinant

We call the Determinant of \(x\) to \(x'\) The number that is the sum of \(2^b\) for all \(b\in B_{x\mapsto x'}\). \[\textrm{det}_{x\mapsto x'}=\sum_{i=0}^{a-1} 2^{b_i}\quad \forall b_i\in B_{x\mapsto x'}\]

Corolaries

  • If you know the determinant then tou can know all of \(b\in B\).
  • If you know all of \(b\in B\) then you can know \(c\)
  • If you know the determinant then you can know \(a\)
