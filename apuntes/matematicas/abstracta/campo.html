Sobre el Campo

Un Conjunto \(K\) bajo dos operaciones, suma y multiplicación, \((K,+,\cdot)\) es un Campo si cumple las siguientes propiedades

  • Es un Grupo Conmutativo con la Suma.
  • Es un Grupo Conmutativo con la Multiplicación (no consistente con 0)
  • La Multiplicación se distribuye en la Suma

Esta definición nos está diciendo que la Suma y la Multiplicación tiene las mismas propiedades, las propiedades de grupo, además la suma y la multiplicación están relacionadas por la Ley distributiva, ahora, ¿Que es un Grupo?

Un Conjunto \(K\) bajo una operación, suma por ejemplo, \((K,+)\) es un

Grupo Conmutativo si cumple las siguientes propiedades

P1 : \(a+b=b+a\)

P2 : \((a+b)+c=a+(b+c)\)

P3 : \(a+0=a\)

P4 : \(a+(-a)=0\)

Donde P1 es la ley conmutativa, P2 la ley distributiva, P3 nos dice que hay un elemento neutro y P4 nos dice que todo elemento tiene un inverso que da el elemento neutro. También existe P0,

Llamada Cerradura que nos dice que si \(a\) y \(b\) pertenecen a \(K\) entonces \(a+b\) pertenece a \(K\), si un Conjunto cumple todas menos P1 entonces es un Grupo no Conmutativo.

Las Propiedades

Juntando toda esta información podemos asegurar que los Reales, al ser un Campo con la suma y la multiplicación, cumplen las siguientes propiedades.

Ley distributiva

LD : \(a\cdot(b+c)=a\cdot b+a\cdot c\)

Propiedades de la igualdad

Si nos queremos poner muy técnicos podemos agregar las propiedades de la igualdad para saber como funcionan.

P1 : \(a=a\)

P2 : Si \(a=b\) entonces \(b=a\)

P3 : \(a=b\) y \(b=c\) entonces \(a=c\)

Donde P1 es la Ley Reflexiva, P2 es la Ley Simétrica y P3 es llamad transitividad. Además ponemos 2 nuevas reglas para relacionar la

Igualdad con la Suma y Multiplicación.

P4 : Si \(a=b\) entonces \(a+c=b+c\)

P5 : Si \(a=b\) entonces \(a\cdot c=b\cdot c\)

Primeras Demostraciones

Demuestra que \(\overline{(\overline{x})}=x\) \(\)\begin{aligned} x\cdot \overline{x} &=1 && \textrm{Inverso de }x\\ \overline{x}\cdot \overline{(\overline{x})} &=1 && \textrm{Inverso de }\overline{x}\\ x\cdot \overline {x}&=\overline{(\overline{x})}\cdot \overline{x} && \textrm{Transitividad}\\ (x\cdot \overline {x})\cdot x&=(\overline{(\overline{x})}\cdot \overline{x})\cdot x && \textrm{Multiplicación de }x\\ x\cdot (\overline {x}\cdot x)&=\overline{(\overline{x})}\cdot (\overline{x}\cdot x) && \textrm{Transitividad}\\ x\cdot 1&=\overline{(\overline{x})}\cdot 1 && \textrm{Elemento Inverso}\\ x&=\overline{(\overline{x})} && \textrm{Elemento Neutro}\\\end{aligned}\(\)

Si \(a=b\) y \(c=d\) entonces \(a+ c=b+ c\) \(\)\begin{aligned} a&=b && \textrm{Hipótesis}\\ a+ c&= b+ c && \textrm{Suma de }c\\ c&=d && \textrm{Hipótesis}\\ c+ b&= d+ b && \textrm{Suma de }b\\ b+ c&= b+ d && \textrm{Conmutación}\\ a+ c&= b+ d && \textrm{Transitividad}\\\end{aligned}\(\)

La no consistencia con 0

Ahora ya tenemos las herramientas para saber por que en un anillo La multiplicación no es un Grupo consistente. Por nuestra definición de

Multiplicación con \(0\) sabemos. \[0\cdot x=0\]

Esto implica que para 2 números cualesquiera \(a,b\) y suponiendo que es un Grupo consistente se cumple. \(\)\begin{aligned} 0\cdot a &= 0 && \textrm{Definición}\\ 0\cdot b &= 0 && \textrm{Definición}\\ 0 &=0\cdot b && \textrm{Ley Reflexiva}\\ 0\cdot a &= 0\cdot b && \textrm{Transitividad}\\ \overline{0}\cdot (0\cdot a) &= \overline{0}\cdot (0\cdot b) && \textrm{Multiplicación de }\overline{0}\\ (\overline{0}\cdot 0)\cdot a &= (\overline{0}\cdot 0)\cdot b && \textrm{Asociatividad}\\ 1\cdot a &= 1\cdot b && \textrm{Elemento Inverso}\\ a &=b && \textrm{Elemetno Neutro}\\\end{aligned}\(\)

Lo que es una contradicción haciendo este sistema no consistente con 0.
